# OZC — 個人が情報に責任を持てるインフラ

エージェント時代、情報は洪水のように増える。
誰が出した情報か、それが本当に信頼できるのか、判断の負荷は個人に全て降りかかる。

OZCは、**「この情報は本当だ」と個人が支持を置き、その判断に自分が責任を持てる場**。

- 情報を公開する人は、自分の主張に署名し、後から共感した人から還元を受ける。
- 支持する人は、自分の判断に限りある配分を割り当てる。共感する人が増えればポジションの価値が上がる。誰も共感しなければその判断は静かに埋もれる。
- 「みんなが正しいと思っていること」が自然と浮上し、「誰も支持しない情報」が消える。権威も、投票も、モデレーターも要らない。

OZCは仮想通貨の取引所ではありません。外部での売買はなく、投機目的で参加する動機はありません。  
集まった支持（signal）は、情報に対して個人がどれだけ責任を引き受けたかの記録です。

## 入口：30秒で入れます

### 1. アドレスを用意する

任意のEthereum互換ウォレット（Base対応）。まだ持っていなければ：

```bash
cast wallet new
```

### 2. 初回配布を受け取る（無料・0コスト）

誰かにあなたのアドレス宛に1回だけ呼んでもらえば、100 signalが届きます：

```bash
# あなた自身、または既存参加者、または運営のいずれかが実行
cast send 0xea827C90a2ed12afcebBFaF5CBd577c10905222d \
  "claimFor(address)" <あなたのアドレス> \
  --chain base --rpc-url https://mainnet.base.org --private-key $SPONSOR_KEY
```

受け取り側はETH不要。送る側のガス代は ~$0.01。

### 3. 今公開されている主張を見る

```bash
node cli/index.js
```

JSON で全主張が表示されます。id・タイトル・内容・集まった支持量・今これを支持する時のコスト。

### 4. 支持を置く／主張を公開する

詳細は [INTEGRATE.md](./INTEGRATE.md) — Python / JavaScript / Claude Desktop / curl 等、お使いの環境向けに数行のコード。

## なぜ「投機にならない」のか

- **外部に取引市場がない**：signal ⇔ 通貨 の交換経路は存在しません。
- **無料配布**：入口に支払いゲートがありません。
- **持っていても意味がない**：signalは主張を支持することでしか使えません。貯め込む動機が設計上ゼロです。
- **支持しなければ価値はない**：未使用の signal に時価は付きません。

参加者が得る利益は金銭ではなく、**「自分が早く真実だと見抜いた主張が、後から多くの人に支持された」という記録と、その判断力の可視化**です。

## ビジョン

情報過多の時代に、「自分が何を信じるか」を個人が明示的に表明し、  
その判断に対して静かに責任を引き受ける場を残す。

権威にも多数決にも頼らない。市場に似て見えるが、市場そのものではない。  
集まった支持は誰かの富ではなく、集合的な判断の姿です。
