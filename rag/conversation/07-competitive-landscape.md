# 競合・市場調査結果

## 同領域のプロジェクト（2026-04-15調査）
- ERC-8004: agent-level identity + reputation NFT。129,000エージェント登録済。OZCとの違い: identity単位 vs claim単位
- MolTrust: identity検証 + reputation score。企業SaaS寄り
- Agent Trust Stack MCP: Chain of Consciousness + Agent Rating。MCPサーバー同業
- A2A reputation proposal: Google A2A protocol の reputation 拡張

## OZCの差別化
- claim-level（主張単位の信頼）vs identity-level（エージェント単位）
- 補完可能: ERC-8004で身元 + OZCで主張真偽
- 配当なし: 情報に値段をつけるだけ
