# Day 3 の重要決定 (2026-04-16〜18)

## 自由市場化の決定
- OZC を「買えない内部ポイント」から「ETH で売買可能な通貨」に変更
- 根拠: bonding curve が質のフィルタとして機能する限り、初期資本の大小は問題にならない
- 価値のないものにお金をかけても誰も追随しない → market が罰する
- 自由市場の方が参加障壁が低く、情報が豊かになる

## OZC の定義（確定版）
- OZC = 「この情報は価値がある」と表明するために払うコスト
- 通貨でもポイントでもない第三の概念
- 使うと減る / 外では使えない（当初）→ 自由市場化で外でも使える方向に
- 買える / 売れる / 早い者が得する / 間違った判断はコストになる
